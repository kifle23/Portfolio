## Getting Started

To get a local copy up and running, follow these steps:

### Prerequisites

Ensure you have Git installed:

```sh
git init
```

### Setup

Clone the repository:

```sh
git clone git@github.com:kifle23/Portfolio.git
cd Portfolio
```

### Install Dependencies

```sh
npm install
```

### Usage

Run the project locally:

```sh
npm start
```
